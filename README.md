# 🚀 Projeto de Portfólio

Este projeto consiste no desenvolvimento de um **website de portfólio profissional**, criado para apresentar habilidades, experiências, projetos e formas de contato de forma moderna, responsiva e acessível.

O projeto foi desenvolvido utilizando tecnologias modernas de front-end com foco em **performance, escalabilidade e boas práticas de desenvolvimento**.

---

# 🛠️ Tecnologias Utilizadas

## Front-end

* React 18
* Vite
* TypeScript
* Tailwind CSS
* Radix UI

## UI / Componentes

* shadcn/ui
* lucide-react
* react-router-dom

## State / Data

* @tanstack/react-query

## Formulários

* react-hook-form
* zod

## Animações & Interatividade

* tsParticles

## Testes

* Vitest

## Controle de versão

* Git
* GitHub

# 🚀 Como Executar o Front-end da Aplicação

## ✅ Pré-requisitos

Instale os seguintes softwares:

* **Node.js (recomendado versão LTS)**
  https://nodejs.org/

* **Git**
  https://git-scm.com/

* **VS Code (opcional, mas recomendado)**
  https://code.visualstudio.com/

O **npm** já vem junto com o Node.js.

# 🛠️ Solução de Problemas (FAQ)

## 1️⃣ PowerShell bloqueando o comando npm

Se aparecer erro relacionado à execução de scripts:

Execute **uma vez** no PowerShell como administrador:

```powershell
Set-ExecutionPolicy RemoteSigned
```

Depois feche e abra o terminal novamente.

---

## 2️⃣ Dependências não instaladas

Se ocorrer erro relacionado a pacotes não encontrados:

```bash
npm install
```

Se persistir:

```bash
rm -rf node_modules
npm install
```

---

## 3️⃣ Porta já em uso

Se o servidor não iniciar porque a porta está ocupada:

```bash
npm run dev -- --port 8080
```
